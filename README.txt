# Система тестирования на Tkinter

Простой GUI-приложение для проведения тестирования, реализованное на Python с использованием библиотеки Tkinter. Приложение состоит из двух вкладок:
- **Преподаватель**: добавление вопросов с вариантами ответов.
- **Студент**: ввод данных (имя, группа), прохождение теста, просмотр результатов и оценки.

## Структура кода

- **Блок логики и данных**  
  - `questions`: список словарей с вопросами, вариантами и индексом правильного ответа.  
  - Функции для добавления вопросов, проверки ответов, сброса теста, навигации между вопросами и вычисления итоговой оценки.  

- **Меню приложения**  
  - Пункты “Файл → Выход” и “Помощь → О программе”.

- **Вкладка “Преподаватель”**  
  - Поле ввода текста вопроса.  
  - Пять полей для вариантов ответа.  
  - Радиокнопки для выбора правильного варианта.  
  - Кнопки “Добавить вопрос” и “Очистить форму”.

- **Вкладка “Студент”**  
  - Форма для ввода имени и группы.  
  - Кнопка “Начать тест” (начинает тестирование при наличии хотя бы одного вопроса).  
  - После старта теста отображается текущий вопрос и варианты ответов.  
  - Кнопки навигации “Назад”, “Далее” (или “Завершить тест” на последнем вопросе).  
  - Окно с результатами: количество набранных баллов и словесная оценка.

- **Цветовые схемы**  
  - Общее окно: `bg_color = "#F0F8FF"` (светло-голубой).  
  - Вкладка “Преподаватель”: `teacher_bg = "#FFF8DC"` (кремовый).  
  - Вкладка “Студент”: `student_bg = "#F5FFFA"` (мятный).

## Использование

### Вкладка “Преподаватель”

1. В поле **“Вопрос”** введите текст нового вопроса.  
2. Заполните **5 вариантов ответов** в соответствующих полях.  
3. Выберите **правильный вариант** с помощью радиокнопок под списком вариантов.  
4. Нажмите **“Добавить вопрос”**.  
   - Если какое-то поле пустое или не выбран правильный ответ, отобразится ошибка.  
   - После успешного добавления появится уведомление, а счетчик вопросов обновится.  
5. Кнопка **“Очистить форму”** сбрасывает все поля ввода (текст вопроса, все варианты и выбор правильного ответа).

Все добавленные вопросы сохраняются только в оперативной памяти: при закрытии программы список вопросов сбрасывается.

### Вкладка “Студент”

1. Введите **Имя** и **Группу** в соответствующие поля.  
2. Нажмите **“Начать тест”**.  
   - Если не заполнены имя/группа или нет ни одного добавленного вопроса, появится сообщение об ошибке.  
3. После старта теста:
   - Отображается первый вопрос и пять вариантов ответов.  
   - Для каждого вопроса требуется выбрать один вариант с помощью радиокнопок.  
   - Кнопка **“Далее”** переводит на следующий вопрос, **“Назад”** возвращает к предыдущему.  
   - На последнем вопросе вместо “Далее” появляется кнопка **“Завершить тест”**.  

> **Важно:** При возврате назад, если предыдущий ответ был верным, балл снимается и при повторном выборе учитывается заново.

4. После завершения теста отображается **экран результатов**:
   - Имя студента, группа.  
   - Количество правильных ответов из общего числа вопросов.  
   - Словесная оценка (см. раздел “Оценивание”).  
   - Кнопка **“Вернуться к входу”** возвращает на начальный экран, сбрасывает результаты и позволяет пройти тест снова.

## Оценивание

Итоговая оценка вычисляется по проценту правильных ответов:
- **≥ 85 %** → “Отлично! (5)”  
- **70 % – 84 %** → “Хорошо! (4)”  
- **50 % – 69 %** → “Удовлетворительно (3)”  
- **< 50 %** → “Неудовлетворительно (2)”

```python
def calculate_grade():
    total = len(questions)
    pct = (score / total) * 100 if total else 0
    if pct >= 85:
        return "Отлично! (5)"
    elif pct >= 70:
        return "Хорошо! (4)"
    elif pct >= 50:
        return "Удовлетворительно (3)"
    return "Неудовлетворительно (2)"
```
